 Student Dropout Prediction

This project predicts whether a student is at risk of dropping out using machine learning based on academic and personal data.
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Data preprocessing and encoding
Feature engineering (Dropout column)
Data visualization
Model training using Logistic Regression

Good accuracy achieved
Model predicts non-dropout students better

How to Run
```bash
pip install -r requirements.txt
```

Run the notebook to see results.


Helps identify at-risk students for early intervention.
